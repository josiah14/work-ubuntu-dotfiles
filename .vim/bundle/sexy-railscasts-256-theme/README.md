# Description

- A 256-color version-ish of the Sexy RailsCasts Vim theme

# Installation

- Copy sexy-railscasts-256.vim into ~/.vim/colors/ (Unix) or vimfiles/colors/ (Windows) as with 
  other color schemes.
- Type "colorscheme sexy-railscasts-256" into your vimrc file.
